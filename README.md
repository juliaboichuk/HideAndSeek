# GoFish

Игра прятки. Предусмотрено 5 соперников, которые рандомно прячуться в комнатах дома. Задача игрока найти всех соперников за минимальное количество шагов.

Особености игры:
- Стороение дома реализовано через статический класс.
- Игроки прячутся рандомно в комнатах. В одной комнате могут спрятаться несколько игроков. 
- Для передвижения по дому используются ключевые слова: 
    - North
    - South
    - East
    - West
    - Northeast
    - Southwest
    - Southeast
    - Northwest
    - Up
    - Down
    - In
    - Out
- В начале каждого хода игрок получает подсказку - в каком направлении можно передвигаться и есть ли в текущей локации места, в которых может прятаться соперник.
- Комманда Check используется для поиска соперников в текущей комнате.
- Комманда "save название_файла" используется для сохранения текущей игры в текстовый файл.
- Комманда "load название_файла" используется для восстановления игры из файла.
